# Focus Timer

A Pomodoro Focus timer app featuring study/break countdown periods, dynamic circular progress SVG tracks, synth buzzer, and task list components.

## Features
- **Study / Break Intervals**: Tab buttons configures Study (25m), Short Break (5m), and Long Break (15m).
- **SVG Stroke Countdowns**: Circular dashboard updates stroke status synchronously.
- **Audio Beep Synthesis**: Invokes sound oscillators without external static file requests.
